## Оператор цикла do while - Цикл с предусловием

```
program summa
real :: sum = 0, slag
k = 0
  slag = 1.0/(k**3 + k**2 + 1)
  do while (slag>0.0005)
    sum = sum + slag
    slag = 1.0/(k**3 + k**2 + 1)
    k = k + 1
  end do
end
```
